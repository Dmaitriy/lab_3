Копирование файлов и деревьев каталогов.

XCOPY источник [назначение] [/A | /M] [/D[:date]] [/P] [/S [/E]] [/V] [/W]
                           [/C] [/I] [/-I] [/Q] [/F] [/L] [/G] [/H] [/R] [/T]
                           [/U] [/K] [/N] [/O] [/X] [/Y] [/-Y] [/Z] [/B] [/J]
                           [/EXCLUDE:file1[+file2][+file3]...] [/COMPRESS]

  источник       Копируемые файлы.
  назначение  Расположение или имена новых файлов.
  /A           Копировать только файлы с установленным атрибутом "Архивный",
               сам атрибут при этом не изменяется.
  /M           Копировать только файлы с установленным атрибутом "Архивный",
               удалять этот атрибут после копирования.
  /D:м-д-г     Копировать файлы, измененные не ранее указанной даты.
               Если дата не указана, заменяются только конечные файлы
               с более ранней датой, чем у исходных файлов.
  /EXCLUDE:файл1[+файл2][+файл3]...
               Список файлов, содержащих строковые значения.  Каждое строковое значение
               должно находиться в этих файлах в отдельной строке. Если любое из
               строковых значений совпадает с любой частью абсолютного пути копируемого файла,
               этот файл не копируется.  Например, если
               указать строковое значение \obj\ или .obj, из копирования
               будут исключены соответственно все файлы из папки obj или все файлы
               с расширением .OBJ.
  /P           Отображать запрос перед созданием каждого нового файла.
  /S           Копировать папки и вложенные папки, кроме пустых.
  /E           Копировать папки и вложенные папки, в том числе пустые.
               Эквивалентно параметрам /S /E. Можно использовать вместе с параметром /T.
  /V           Проверять размер каждого нового файла.
  /W           Предложить нажать клавишу перед копированием.
  /C           Продолжить копирование даже при возникновении ошибок.
  /I           Если места назначения не существует, а копируется сразу несколько файлов,
               предполагается, что место назначения является папкой.
  /-I          Если место назначения не существует и копируется один указанный файл,
               предполагается, что это место назначения должно быть файлом.
  /Q           Не отображать имена файлов при копировании.
  /F           Отображать полные имена исходных файлов и файлов назначения при копировании.
  /L           Отображать копируемые файлы.
  /G           Разрешить копирование зашифрованных файлов в место назначения, где
               не поддерживается шифрование.
  /H           Также копировать скрытые и системные файлы.
  /R           Перезаписывать файлы, доступные только для чтения.
  /T           Создать структуру папок, но не копировать файлы. Пустые
               папки и вложенные папки не включаются. Чтобы создать
               пустые папки и вложенные папки, используйте параметры /T /E.
  /U           Копировать только файлы, уже существующие в папке назначения.
  /K           Копировать атрибуты. Обычно при использовании команды XCOPY атрибут "Только для чтения" сбрасывается.
  /N           Использовать короткие имена при копировании.
  /O           Копировать данные о владельце файла и сведения списка управления доступом.
  /X           Копировать параметры аудита файлов (требуется параметр /O).
  /Y           Не выводить запрос на подтверждение перезаписи
               существующих файлов назначения.
  /-Y          Обязательно запрашивать перезапись
               существующих файлов назначения.
  /Z           Копировать сетевые файлы с возможностью возобновления.
  /B           Копировать символьную ссылку вместо целевого объекта этой ссылки.
  /J           Копировать с использованием ввода-вывода без буферизации.
  /COMPRESS    Запросить сетевое сжатие при передаче файла во всех
               возможных случаях.
  /SPARSE      Сохранить разреженное состояние при копировании разреженного файла.

Параметр /Y может быть предустановлен в переменной среды COPYCMD.
